# A/B Testing for HR Interventions

## Overview
This project shows a simple experimental design workflow for comparing a control group and an intervention group in an HR setting.

## Files
- `ab_test_sample.csv`
- `ab_test_example.py`
- `ab_test_boxplot.png`

## Current synthetic result
- Mean difference: **2.72**
- 95% CI: **(1.29, 4.16)**
- p-value: **0.0002**

## Business framing
A/B testing helps HR teams evaluate whether a training, onboarding, or manager enablement intervention had a measurable effect before scaling it broadly.
