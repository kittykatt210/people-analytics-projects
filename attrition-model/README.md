# Attrition Risk Model

## Overview
This portfolio project demonstrates how a people analytics team might estimate voluntary attrition risk and identify the strongest synthetic drivers of turnover.

## Files
- `synthetic_attrition_data.csv`
- `attrition_model.py`
- `attrition_drivers.png`

## Methods
- Logistic regression
- Random forest classification
- Basic feature engineering and preprocessing
- Model comparison using accuracy and AUC

## Current synthetic benchmark
- Logistic regression accuracy: **0.934**
- Logistic regression AUC: **0.657**
- Random forest accuracy: **0.931**
- Random forest AUC: **0.737**

## Business framing
This project is designed to show how analytics can support:
- early risk detection
- manager enablement
- targeted retention strategies
- people-centric decision support

## Resume bullet
