# Microsoft People Analytics Portfolio Starter

## Included projects
1. **Attrition risk modeling**
   - Synthetic employee dataset
   - Logistic regression and random forest starter code
   - Driver chart
   - Current synthetic benchmark: logistic AUC **0.657**, random forest AUC **0.737**

2. **Workforce dashboard mockups**
   - KPI sample dataset
   - Headcount, engagement, and attrition trend visuals
   - Python script to regenerate charts

3. **A/B testing for HR interventions**
   - Sample experiment dataset
   - Statistical test starter script
   - Boxplot for presentation or README use

## Important note
These projects are provided as **portfolio-ready examples** and use **synthetic sample data** so they can be safely shared publicly without exposing confidential employer information.
